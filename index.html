Xcode和Mac 常用快捷键

1，移动代码的快捷键: 上下移动: command + option + [(上) / ](下)

左右移动: command + ](右) / [(左)

2，展开/隐藏 代码的快捷键:

command + option + <— （隐藏）

command + option + —>（展开）

3，全局修改局部变量快捷键:

command + control + e

4，弹出方法列表

control + 6

5，隐藏/打开控制台

command + shift + y

6，快速打开全局查询

command + shift + f

7，快速切换到.m文件/.h文件:

command + control + up/down

8，快速切换到文件的顶部/底部:

command + up/down

9，删除光标所在行之前的代码:

command + delete

10，将光标移动到所在行的最前面/后面:

command + 左 / 右

11，快速切换到文件的顶部/底部:

command + up/down

MAC操作

1，终端新建多个窗口

command+T

2，多个终端窗口之间的切换。

Command + shift + {或}进行左右切换标签。
